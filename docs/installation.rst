====================
Installation
====================

Pre-Build
==========

.. code-block:: bash

    git clone http://github.com/ehouse/mirrors.git
    ./setup.py build

Install
==========
Running in a virtualenv isn't required but recommended. 

.. code-block:: bash

    ./setup.py install

Uninstall
==========

.. code-block:: bash

    pip uninstall mirrors
